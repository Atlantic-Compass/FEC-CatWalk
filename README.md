# projectcatwalk

A new e-commerce site for our client made by Team Atlantic Compass

## Building and running on localhost

First install dependencies:

```sh
npm install
```

Build for production:

```sh
npm run build
```

## Running

Open the file `dist/index.html` in your browser

## Testing

To run unit tests:

```sh
npm test
```
